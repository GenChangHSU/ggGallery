Introduction

Bump charts are a special type of line charts that visualize the rankings of multiple objects over time. These charts are especially useful for comparing and highlighting trends. Honestly, I haven’t really heard of this kind of charts before until recently I came across a Tweet about creating bump charts using ggplot. I then took a deeper dive into it and felt that this would certainly make an interesting topic to blog about. Also, I believe the charts will come in handy some day in the future, both for myself and for the reader like you!

So without further ado, let’s jump right in!

Prepare the data

As mentioned earlier, bump charts are useful for visualizing rankings of items over time. Instead of using a built-in R dataset, this time I’ll prepare my own one, which focuses on the rankings of five ecological journals (Journal of Ecology, Journal of Animal Ecology, Journal of Applied Ecology, Functional Ecology, and Methods in Ecology and Evolution) from the British Ecological Society. Specifically, I would like to see how the rankings of these five journals, in terms of their impact factors, change over the past decade.

To get the data, I visited Journal Citation Reports website and downloaded their impact factor records.

Let’s read in the datasets and do some data wrangling:

library(tidyverse)
library(janitor)  # for the function "row_to_names()"

# Create a list of url paths
BES_journals <- c("J_ECOL", "J_ANIM_ECOL", "J_APPL_ECOL", "FUNCT_ECOL", "METHODS_ECOL_EVOL")
dataset_urls <- str_glue("https://raw.githubusercontent.com/GenChangHSU/ggGallery/master/_posts/2022-08-01-post-18-bump-charts-with-ggplot/BES_journals_data/{BES_journals}.csv")

# Loop through the url paths
BES_journals_data <- map(dataset_urls, function(x){
  read_csv(x) %>% 
    row_to_names(row_number = 5) %>%  # set the fifth row as column names
    slice_head(n = nrow(.)-4)  # remove the last four rows
  }) %>%
  `names<-`(BES_journals) %>%
  bind_rows(.id = "Journal") %>%  # convert the list to a dataframe
  select(Journal, Year, IF = `Journal impact factor`) %>% 
  filter(Year >= 2011) %>%  # retain data starting from 2011
  group_by(Year) %>% 
  mutate(IF = as.numeric(IF),
         Ranking = row_number(desc(IF))) %>%  # add the journal rankings by year
  arrange(Year, Journal)

library(DT)  # for the function "datatable()"
datatable(BES_journals_data, options = list(pageLength = 5))

Basic bump chart

Now we have our data at hand, it’s time for the chart!

Creating a basic bump chart is as simple as making a scatterplot (or dotplot) of time vs. rankings and “connecting the dots” with lines, which can be done with geom_point() and geom_line(). Next, we can use scale_y_reverse() to flip the y-axis so that the highest-ranking item stays on the top of the plot.

# A color palette for the journals
col_pal <- set_names(c("#0e96d4", "#7581bd", "#48a749", "#e38d26", "#de2127"), nm = BES_journals)

# The basic bump chart
ggplot(BES_journals_data) + 
  geom_line(aes(x = Year, y = Ranking, color = Journal, group = Journal), size = 2) + 
  geom_point(aes(x = Year, y = Ranking, color = Journal), shape = 21, fill = "white", size = 4, stroke = 2) +
  scale_y_reverse() +  # the highest-ranking journal is on the top
  scale_color_manual(values = col_pal) +  # use the customized color palette
  theme_classic()

Seems that Methods in Ecology and Evolution has been the leading journal of the five in recent years!

Advanced topic - Ribbon bump chart

For the purpose of science communication, the bump charts we’ve created above are pretty much enough. But as a ggplot geek, it’s always fun to challenge myself a bit and go a step further to try out something cool. This is the motivation for this advanced topic! Also, a shout-out to the author of the Tweet I mentioned at the beginning of the post. The idea was largely inspired by the chart he made and much of the code below is modified from his GitHub repository.

Creating a ribbon bump chart is not an easy task and requires some data manipulations. I’ll break down the process into four steps and explain what I’m doing in each step. I think this would make it easier to understand the underlying principles and hopefully you’ll be able to build your own chart in the future!

So the four steps are: (1) Create a stacked barplot of journals’ impact factors each year (2) Compute the curves that connect the upper/lower ends of the bars for each journal (3) Add ribbons to the stacked bar plot (4) Polish the appearance of the chart

Let’s kick off the journey and see where we end!

Step 1. Create a stacked barplot of journals’ impact factors over year

To create a stacked barplot of journals’ impact factors, we need to determine the upper and lower ends of the bars. We’ll do this by first arranging the journals in a descending order based on their rankings each year (so the lowest-ranking journal will be on the top of the dataframe) and then calculating the cumulative sums of the journals’ IFs, which will be the upper ends of the bars. For the lower ends of the bars, they’ll be the cumulative sums minus the journals’ own IFs.

It’s noteworthy that here we did not use the conventional geom layer geom_bar() to create the barplot, but instead we used geom_rect(). The main reason for using geom_rect() rather than geom_bar() is that geom_rect() allows us to order the bars differently for each year (since the journal rankings differ across years), whereas geom_bar() will place the bars in the same order across years.

# Calculate the cumulative sums of the journals' IF 
BES_journals_data_barplot <- BES_journals_data %>% 
  arrange(Year, desc(Ranking)) %>% 
  group_by(Year) %>% 
  mutate(y_upper = cumsum(IF),
         y_lower = cumsum(IF) - IF) %>% 
  ungroup()

# A color palette for the journals
col_pal <- set_names(c("#0e96d4", "#7581bd", "#48a749", "#e38d26", "#de2127"), nm = BES_journals)

# Create a stacked barplot of the journals' IFs each year
ribbon_bumpchart_bars <- ggplot(BES_journals_data_barplot) + 
  geom_rect(aes(xmin = as.numeric(Year) - 0.05, xmax = as.numeric(Year) + 0.05,
                ymin = y_lower, ymax = y_upper, fill = Journal), color = "white") +
  scale_fill_manual(values = col_pal) +
  theme_classic()

ribbon_bumpchart_bars

Step 2. Compute the curves that connect the upper/lower ends of the bars for each journal

Now comes the core part of the process: computing the curves that connect the upper/lower ends of the bars for each journal. These upper and lower curves form the boundaries of the ribbons that we’ll be adding to the plot later.

So for each journal, we’ll need the curves that connect the upper/lower ends of the bars. To get the curve, we’ll create four new columns each year: x_from, x_to, y_from, and y_to. x_from is the current year plus the bar width; x_to is the next year minus bar width, y_from is the upper/lower ends of the bars in current year, y_to is the upper/lower ends of the bars next year.

Also, I subtracted 0.1 from the upper boundary and added 0.1 to the lower boundary to create a margin between ribbons so that they will not get sticked to each other.

for each of the two adjacent years, we’ll have two points, which will be x_from, x_to, y_from, and y_to. x_from and x_to are simply the years plus/minus the bar width. y_from, and y_to are the upper/lower ends of the two adjacent bars.

The y_to for the year 2022 was set to NA because there is no further bars to go from, and later the rows for 2022 were removed.

Finally, I used the function sigmoid() from the package ggbump to estimate a smooth sigmoid curve between the two points.

library(ggbump)  # for the function "sigmoid()"

# Compute the upper and lower curves connecting the bars
BES_journals_data_ribbon <- BES_journals_data_barplot %>% 
  split(., .$Journal) %>% 
  map(., function(data){
    
    # the upper boundary
    upper_curve <- data %>% 
      select(Journal, Year, y_upper) %>% 
      mutate(x_from = as.numeric(Year) + 0.05,
             x_to = as.numeric(Year) + 1 - 0.05,
             y_from = y_upper - 0.1,  # 0.1 sets the margin between the adjacent curves
             y_to = c(y_upper[-1], NA) - 0.1) %>% 
      filter(Year != "2021") %>% 
      rowwise() %>% 
      mutate(curve = list(sigmoid(x_from, x_to, y_from, y_to, n = 100, smooth = 3))) %>% 
      unnest() %>% 
      select(Journal, x, y_upper = y)
    
    # the lower boundary    
    lower_curve <- data %>%
      select(Journal, Year, y_lower) %>% 
      mutate(x_from = as.numeric(Year) + 0.05,
             x_to = as.numeric(Year) + 1 - 0.05,
             y_from = y_lower + 0.1,
             y_to = c(y_lower[-1], NA) + 0.1) %>% 
      filter(Year != "2021") %>% 
      rowwise() %>% 
      mutate(curve = list(sigmoid(x_from, x_to, y_from, y_to, n = 100, smooth = 8))) %>% 
      unnest() %>% 
      select(Journal, x, y_lower = y)
    
    # put the two dataframes together
    curve <- left_join(upper_curve, lower_curve[, -1], by = "x")
  }) %>% bind_rows()

datatable(BES_journals_data_ribbon, options = list(pageLength = 5))

Step 3. Add ribbons to the stacked bar plot

# Add the ribbons to the bar plot 
ribbon_bumpchart <- ribbon_bumpchart_bars + 
  geom_ribbon(data = BES_journals_data_ribbon, 
              aes(x = x, ymin = y_lower, ymax = y_upper, fill = Journal), alpha = 0.75)

# Move the bars to the top  
library(gginnards)

ribbon_bumpchart <- move_layers(ribbon_bumpchart, "GeomRect", position = "top")
